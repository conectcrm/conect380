# Catalogo de Regras de Negocio

Este diretorio concentra as regras de negocio que precisam ser preservadas em qualquer alteracao de codigo (humana ou por agente de IA).

## Como usar

1. Identifique a regra afetada pelo seu PR.
2. Atualize o mapeamento `regra -> implementacao -> testes`.
3. Referencie os IDs das regras no PR.
